# 🌌 stars_reader

An initial study utilizing **Diffusion-based methods** and **Generative Adversarial Networks (GANs)** to decode hidden messages and signs in the stars from an earth-sky perspective.
---

## 📋 Project Overview
`stars_reader` explores the intersection of deep learning and celestial observation. By applying state-of-the-art denoising and generative architectures, this study aims to isolate foreground signals and apply cryptographic decoding techniques to astronomical datasets, seeking the "signs" (Ayat) hidden within the cosmic horizon.
